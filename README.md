## 📌 Introduction

**ZyroMart** is a modern and responsive e-commerce frontend application built using **React** and **JavaScript**.

The application allows users to browse products, view detailed information, manage orders, and seamlessly interact with a backend API.

This project focuses on:

- Clean UI/UX
- Scalable component architecture
- REST API integration
- Efficient state management
- Fully responsive design

## ⚡ Quick Start

Follow these steps to run the project locally.

### 📌 Prerequisites

Make sure you have:

- Node.js (v14+ recommended)
- npm installed
- ZyroMart backend server running

---

### 📥 Clone the Repository

```bash
git clone https://github.com/Sakib-Atreus/ZyroMart-client
cd ZyroMart-client
```

### 📦 Install Dependencies

```bash
npm install
```

### ▶️ Start Development Server

```bash
npm start
```
